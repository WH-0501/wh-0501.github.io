新建项目

创建项目目录 D:\workspaces\Project_test，新建项目

Keil 5 Project -> New uVision Project，定位到刚创建的 USER 目录，然后选择相应 CPU 型号（此处以 STM32F103C8T6 为例）

若列表中只有 ARM，没有 STM32 系列，则通过 Pack Installer 安装相应包

在 RTE 窗口中选择 CORE 及 Startup

在该目录下分别创建 CORE、OBJ、STM32_F103x_FWLIB、SYSTEM、USER

创建 CMSIS、FWLIB、Project、Startup、User 目录。

文件夹名称 存放内容
CMSIS 内核相关的文件
FWLIB 标准库文件
Project 工程文件
Startup 芯片启动文件
User 用户编写的相关文件，主要编写的文件都在这个文件夹里
Doc 对整个工程项目的补充说明、介绍和简介

下载标准库及启动文件

CMSIS 目录

从 CubeMX 生成的项目中拷贝过来。

FWLib

存放 ST 标准库文件。将 STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\STM32F10x_StdPeriph_Driver 中的 inc 及 src 目录拷贝进来即可。
