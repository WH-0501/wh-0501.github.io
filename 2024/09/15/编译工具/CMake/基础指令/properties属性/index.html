SET_TARGET_PROPERTIES

常用属性包括：

  • CXX_STANDARD/C_STANDARD : 设置 C/C++ 版本
  • CXX_EXTENSIONS/C_EXTENSIONS : 设置是否启用 C/C++ 标准库扩展
  • OUTPUT_NAME : 设置生成的文件的名称
  • RUNTIME_OUTPUT_DIRECTORY : 设置文件运行时的输出目录
  • INCLUDE_DIRECTORIES : 设置目标的包含目录
  • COMPILE_DEFINITIONS : 为目标设置编译时宏定义

GET_TARGET_PROPERTY

从 target 获取属性。

该属性的值存储在变量 <var> 中。如果没有找到目标属性，则行为取决于它是否被定义为 INHERITED 属性(参见 define_property())。非继承属性将 <var> 设置为 <var>-NOTFOUND，而继承属性将按照 define_property() 命令的描述搜索相关的父范围，如果仍然无法找到该属性，则将 <var> 设置为空字符串。

使用 set_target_properties() 设置 target 属性值。属性通常用于控制如何构建 target，但有些则用于查询 target。该命令可以获取到目前为止创建的任何 target 的属性。target 不需要在当前的 CMakeLists.txt 文件中。
